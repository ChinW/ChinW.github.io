  • 'Realm/RLMArray.h' file not found

    9月 28, 2016 in iOS

    错误症状

    模拟器运行正常，打包正常，真机编译时部分Framework出现

    ‘Realm/RLMArray.h’ file not found

    背景

    CocoaPods 1.0.1, Xcode 8, Realm 0.98.3

    解决方式

    按下Option+CMD+Shift+K, Clean Build Folder, 或者按下Option，选择｀Product｀—> Clean Build Folder
